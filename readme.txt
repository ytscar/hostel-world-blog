Base repo & plugin for fixing bugs and resolving issues in the blog website.

== Installation ==

1. Go to 'Plugins > Add New' on your WordPress admin dashboard.
2. Search for 'Hostel World Blog' plugin from the official WordPress plugin repository.
3. Click 'Install Now' and then 'Activate'.
